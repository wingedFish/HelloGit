1、创建仓库
	在github.com首页右小脚点击 new repository

2、初始化新的版本库
	本地代码文件下右键选择GitExt Create New Repository
	选择需要初始化的目录
	初始化成功后，会生成.git文件夹。里面conf是配置文件，包含远程路径

3、客户端add\commit\push
	commit时可选择 new branch ，则自动创建新的分支。有问题则留着
	push时 Destination 下的 Remote,选择manager设置远程名字
	

4、创建分支
	文件中右键--Git Exsisens--create brach
	
5、提交分支测试
